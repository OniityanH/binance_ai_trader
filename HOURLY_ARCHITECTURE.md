# AI Trader 架构设计 (每小时决策版)

## 🎯 交易币种
- **BTC** (比特币)
- **ETH** (以太坊)
- **SOL** (Solana)
- **BNB** (币安币)
- **XRP** (瑞波币)
- **DOGE** (狗狗币)

---

## ⏰ 执行频率
- **每 1 小时执行一次**
- 每次对 6 个币种各做一次决策

---

## 📊 数据获取策略

### 1. 价格数据 (Binance API)

| 数据类型 | 时间范围 | 用途 |
|---------|---------|------|
| 实时价格 | 当前 | 判断买入/卖出 |
| 1小时K线 | 最近 24 根 (24小时) | 短期趋势 |
| 15分钟K线 | 最近 16 根 (4小时) | 精细趋势 |
| 订单簿 | 当前 | 流动性分析 |
| 资金费率 | 当前 | 多空情绪 |

### 2. 鲸鱼/大户数据

| 数据类型 | 时间范围 | 来源 |
|---------|---------|------|
| 大额转账 | 最近 1 小时 | WhaleAlert (可选) |
| 交易所资金流向 | 最近 24 小时 | CryptoQuant (可选) |
| 持币地址分布 | 当前 | IntoTheBlock (可选) |

### 3. 新闻情绪

| 数据类型 | 时间范围 | 来源 |
|---------|---------|------|
| 最新新闻 | 最近 6 小时 | CryptoCompare |
| 情绪分析 | AI判断 | 关键词分析 |

## 🧠 DeepSeek 输入设计

### System Prompt (不变)
```
你是一个专业的虚拟货币交易分析师。你的任务是分析市场数据并做出买入/卖出/持有决策。

核心原则:
1. 永远把风险控制放在第一位
2. 只在高置信度时下单
3. 严格执行止损策略
4. 不追涨杀跌

你必须输出JSON格式的决策。
```

### User Prompt (每币种)

```
## 交易对
{symbol} (如 BTCUSDT)

## 持仓状态
- 当前持仓: {position} {symbol}
- 开仓价格: ${entry_price}
- 当前盈亏: {unrealized_pnl}%
- 持仓时间: {holding_hours}小时

## 价格数据
- 当前价格: ${current_price}
- 24h涨跌: {price_change_percent}%
- 24h成交量: ${volume}

## 技术分析

### 1小时K线 (最近6根)
- 06:00: 67000 → 67200 (上涨)
- 05:00: 66800 → 67000 (上涨)
- 04:00: 67100 → 66800 (下跌)
- 03:00: 66900 → 67100 (上涨)
- 02:00: 66500 → 66900 (上涨)
- 01:00: 66200 → 66500 (上涨)

### 15分钟K线 (最近4根)
- 10:45: 67000 → 67100
- 10:30: 66900 → 67000
- 10:15: 66800 → 66900
- 10:00: 66700 → 66800

### 均线 (可选)
- MA20: {ma20}
- MA60: {ma60}

### 订单簿
- 买方深度: $500万
- 卖方深度: $300万
- 买卖价差: $5

### 资金费率
- 当前费率: 0.01% (多头略占优)

## 链上/鲸鱼数据
- 24h大额转账: 5笔 (总计$5000万)
- 交易所净流入: +$2000万 (健康)
- 巨鲸持仓变化: +1.2%

## 新闻情绪
- 近期新闻:
  1. "机构比特币ETF净流入创新高"
  2. "以太坊升级获社区支持"
- 综合情绪: bullish

## 风控参数
- 最大止损: 7%
- 止盈目标: 15%
- 风险偏好: 保守
```
